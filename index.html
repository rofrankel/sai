Sai.JS - a CoffeeScript charting library

Sai is a vector charting library written in CoffeeScript and using Raphael.JS for cross-browser rendering (VML in IE<8, SVG elsewhere). It is current not open source, though this may change in the future.

Following are some examples of the kinds of charts Sai can be used to create.

Line

The basic type of chart supported by Sai is the line chart. 

<div id="line_chart_1" />
<script type="text/javascript">
    var r1 = Raphael("line_chart_1", 350, 350);
    var colors = {
        Alice: 'grey',
        Bob: 'blue',
        Carol: '#AF280D',
        Dave: '#c00',
    };
    var data = {
        Alice: [5, 2, 5, 1, 2, 6, 2],
        Bob: [7, 3, 6, 12, 9, 4, 3],
        Carol: [3, 2, 5, 2, 5, 2, 1],
        Dave: [1, 1, 4, 5, 6, 5, 2],
        Month: ['January', 'February', 'March', 'April', 'May', 'June', 'July', 'August']
    };
    var chart = new Sai.LineChart(r1, 0, 350, 350, 350, data, "Month");
    chart.setColors(colors);
    chart.render();
    // instaed of the previous three lines, the following would also work:
    // var chart = new Sai.LineChart(r1, 0, 350, 350, 350, data, "Month").setColors(colors).render();
</script>
