Sai.JS - a JavaScript charting library

Sai is a vector charting library written in CoffeeScript and using Raphael.JS for cross-browser rendering (VML in IE<8, SVG elsewhere). It is current not open source, though this may change in the future.

Following are some examples of the kinds of charts Sai can be used to create.

Line

The basic type of chart supported by Sai is the line chart. 

<!-- create a holder div for the Raphael canvas-->
<div id="line_chart_1" />
<script type="text/javascript">
    var r1 = Raphael("line_chart_1", 350, 350);
    var options = {
        bgcolor: '#EEE',
        interactive: true,
        groups: {
            left: ['Michael', 'Eliezer', 'Jasen'],
            right: ['Sean']
        }
    };
    var colors = {
        Michael: 'grey',
        Eliezer: 'blue',
        Jasen: 'green',
        Sean: 'yellow',
        __RIGHTAXIS__: 'red'
    };
    var data = {
        Michael: [5, 2, 5, 1, 2, 6, 2],
        Eliezer: [7, 3, 6, 12, 9, 4, 3],
        Jasen: [3, 2, 5, 2, 5, 2, 1],
        Sean: [1, 1, 4, 5, 6, 5, 2],
        Month: ['January', 'February', 'March', 'April', 'May', 'June', 'July', 'August']
    };
    var chart = new Sai.LineChart(r1, 0, 350, 350, 350, data, "Month", options);
    chart.setColors(colors)
    chart.render();
</script>
